# Louis Wang's Year Planner

**Version:** v1.0.0 | **Status:** Live & Active | **License:** MIT | **Author:** Louis Wang

---

A fast, visual Gantt chart year planner built with **React** and **Tailwind CSS**. No sign-up, no accounts  everything saves instantly in your browser.

---

## Overview

**Louis Wang's Year Planner** is a personal project born from a simple need: a clean, powerful, and beautiful way to plan an entire year at a glance.

Built entirely with React and Tailwind CSS, this planner delivers a fully interactive Gantt chart experience  right in the browser, with zero backend dependency. It reflects a commitment to clean UI, smooth interactions, and thoughtful user experience design.

Though built as a personal tool, it stands as a complete, production-quality application  a testament to what focus and passion can produce.

---

## Project Vision

In a world full of overcomplicated project management tools, **Louis Wang's Year Planner** was created to:

- Offer a **clean, distraction-free** planning experience
- Make **drag-and-drop timeline management** feel effortless
- Support **multiple planners** for different projects or goals
- Run entirely **offline**  no accounts, no cloud, no friction
- Demonstrate what a **single developer** can build with modern web tools

---

## Features

| Feature | Description |
|---------|-------------|
| Multiple Planners | Create separate planners for different projects or years |
| Drag and Drop | Move and resize task bars directly on the timeline |
| Undo / Redo | Full history so you can experiment freely |
| Dark / Light Mode | Toggle between themes, persisted across sessions |
| Accent Colours | Choose from Blue, Purple, Green or Orange themes |
| Colour Coded Tasks | Assign colours to tasks for instant visual grouping |
| Task Notes | Attach notes to any task for extra context |
| Zoom Controls | Zoom slider, Ctrl+scroll, and fit-to-view |
| Import / Export | Back up or share planners as JSON files |
| Google Translate | Built-in language switcher in the navbar |

---

## Tech Stack

| Component | Description |
|-----------|-------------|
| **React 18** | UI & component-based state management |
| **Tailwind CSS 3** | Utility-first styling |
| **localStorage** | Zero-backend data persistence |
| **Pointer Events API** | Smooth drag & drop interactions |
| **Express (optional)** | Lightweight backend API on port 3001 |
| **lucide-react** | Clean, consistent icon library |

## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/Louisw123451/Louis-Wang-s-Year-Planner.git
cd Louis-Wang-s-Year-Planner
```

2. **Install dependencies:**

```bash
npm install
```

3. **Start the dev server:**

```bash
npm start
```

4. Open http://localhost:3000 in your browser.
